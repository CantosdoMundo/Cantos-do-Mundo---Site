# Cantos do Mundo - Site

Site oficial da Cantos do Mundo – Experiências e Turismo.

## Como Iniciar (Desenvolvimento)

Primeiro, instale as dependências e rode o servidor de desenvolvimento:

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.

## Deploy

Para colocar online, rode:

```bash
npm run build
```

E arraste a pasta `out` para o Netlify Drop.
